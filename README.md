# Postal Forecast

A standalone weather app that looks up a postal code, converts it to coordinates, and renders a 5-day forecast.

## Run

Open `index.html` in a browser, or serve this folder locally:

```bash
python3 -m http.server 5174
```

Then visit `http://127.0.0.1:5174`.

## APIs

- Postal lookup: Zippopotam.us
- Forecast data: Open-Meteo
